使用ReSeqTools/iTools进行fastq数据统计、质检、过滤

2015-01-02 | Bioinformatics

重测序分析集成工具包 linux 64位平台 静态编译 chmod 便可以直接运行了。 （ReSeqTools/iTools） 即在linux64机上只须

  • tar -zxvf ReSeqTools_XXX.tar.gz ;
  • cd iTools_Code; chmod 775 iTools ;
  • ./ iTools -h ;
  • 无须重新make
